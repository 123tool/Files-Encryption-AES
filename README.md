**Files Encryption AES** adalah aplikasi berbasis web modern yang dirancang untuk melindungi privasi data Anda dengan enkripsi kelas militer (AES-256). Tool ini dikembangkan oleh **123Tool & SPY-E** untuk memberikan solusi keamanan yang cepat, transparan, dan sepenuhnya berjalan di sisi klien (Client-Side).

---

## 🚀 Features

- **Advanced AES-256 Encryption:** Menggunakan algoritma enkripsi standar industri yang sangat kuat.
- **Zero-Server Exposure:** Data Anda tidak pernah dikirim ke server. Seluruh proses enkripsi dan dekripsi terjadi langsung di browser Anda.
- **Custom Extension (.spy-e):** Mengamankan file dengan format khusus milik ekosistem SPY-E.
- **Lightweight & Fast:** Dibangun dengan optimasi tinggi agar dapat berjalan lancar bahkan di perangkat berspesifikasi rendah melalui browser.

---

## 🛠️ Tech Stack

- **Core:** JavaScript (ES6+)
- **Security Library:** CryptoJS (AES Engine)
- **Styling:** CSS3 dengan efek Glassmorphism & Animations
- **Typography:** Orbitron & Inter (Google Fonts)

---

## 📖 How to Use

1. **Select Mode:** Pilih "Encrypt" untuk mengunci file atau "Decrypt" untuk membuka file.
2. **Load File:** Masukkan file yang ingin diproses (Dokumen, Gambar, atau Database).
3. **Set Password:** Masukkan *passphrase* rahasia Anda. **Peringatan:** Jika password lupa, file tidak akan pernah bisa dibuka kembali!
4. **Execute:** Tekan tombol execute dan unduh hasil file Anda yang sudah terproteksi.
